What is B4AE?

B4AE is a quantum-ready secure transport abstraction layer — not a Signal competitor, but infrastructure that layers under Signal, Matrix, MQTT, gRPC, and any app needing quantum-resistant encryption.

  • Add quantum security in 5 minutes
  • Pluggable, modular, audit-friendly
  • Kyber-1024, Dilithium5 (NIST-standardized PQC)

Quick Start

cargo add b4ae --features elara

Full installation →

Features

  • Quantum-resistant: Kyber-1024, Dilithium5, hybrid with X25519/Ed25519
  • Metadata protection: Padding, timing obfuscation, dummy traffic
  • Enterprise: AuditSink, compliance mapping (GDPR, HIPAA, ISO 27001)
