# Whitepaper Publication Checklist

Publikasi whitepaper ke IACR ePrint, arXiv, dan komunitas NIST PQC.

---

## 1. Draft Whitepaper

### Konten Minimal
- [ ] Ringkasan eksekutif
- [ ] Deskripsi protokol (handshake, message format)
- [ ] Cryptographic building blocks (Kyber, Dilithium, hybrid)
- [ ] Metadata protection (padding, timing, dummy traffic)
- [ ] Security assumptions & threat model
- [ ] Performance benchmarks (dengan metodologi)
- [ ] Comparison dengan E2EE / TLS PQ

### Referensi
- [Protocol Specification](../specs/B4AE_Protocol_Specification_v1.0.md)
- [PERFORMANCE.md](PERFORMANCE.md)
- NIST PQC standardization docs

---

## 2. IACR ePrint

- [ ] Buat akun di https://eprint.iacr.org/
- [ ] Format: PDF, LaTeX preferred
- [ ] Submit ke kategori: Cryptography and Security
- [ ] Dapatkan ePrint ID (e.g. 2025/xxx)
- [ ] Tambahkan link di README & docs

---

## 3. arXiv

- [ ] Buat akun di https://arxiv.org/
- [ ] Submit ke cs.CR (Cryptography and Security)
- [ ] Referensi ePrint jika sudah ada
- [ ] Tambahkan arXiv ID di README

---

## 4. Visibility

- [ ] Share di NIST PQC mailing list / community
- [ ] Crypto StackExchange (Q&A, visibility)
- [ ] Twitter/X, LinkedIn (jika ada akun)
- [ ] Reddit r/crypto (sesuai aturan sub)

---

## 5. Post-Publication

- [ ] BibTeX citation di README
- [ ] Update [STRATEGIC_VISION.md](STRATEGIC_VISION.md) dengan link
- [ ] Announce di GitHub Discussions / Release notes
